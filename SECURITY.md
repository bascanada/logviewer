# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. 

Instead, please report it via [GitHub Security Advisories](https://github.com/bascanada/logviewer/security/advisories/new).

We take all security vulnerabilities seriously and will respond promptly to valid reports.

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |

## Security Best Practices

When using logviewer:
- Keep your installation up to date with the latest version
- Review and restrict access credentials for log sources
- Use environment variables or secure configuration management for sensitive data
- Follow the principle of least privilege when granting access permissions
